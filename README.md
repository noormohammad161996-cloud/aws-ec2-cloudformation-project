# 🚀 AWS S3 Static Website using CloudFormation

This project demonstrates how to deploy a static website on AWS S3 using Infrastructure as Code (IaC) with CloudFormation.

---

## 📌 Project Overview

In this project, I created an automated infrastructure setup using AWS CloudFormation to:

- Create an S3 bucket
- Enable static website hosting
- Upload HTML & CSS files
- Configure bucket policy for public access
- Access the website using S3 endpoint

---

## 🛠️ Technologies Used

- AWS S3
- AWS CloudFormation
- HTML5
- CSS3
- Git & GitHub
